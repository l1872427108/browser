    # 页面生命周期

    我们来看一看从浏览器下载回资源，到浏览器卸载会触发几个浏览器的生命周期事件。

    # DOMContentLoaded

    浏览器已经加载完毕 HTML，但是资源还没有加载完毕，例如：img，css外部w资源.

    # load

    浏览器不仅加载完毕 HTML，而且加载完毕所有的资源文件。

    # readyState

    document 的一个属性，它提供了当前加载状态的信息。

    • loading 文档正在被加载。
    • interactive 文档被全部读取
    • complete 文档被全部读取, 包括其他资源。

    readystatechange 事件，会在状态发生变化时触发。

    加载顺序：

    1. loading
    2. interactive
    3. DOMContentLoaded
    4. 外部资源加载完毕 img onload
    5. complete
    6. onload

    # beforeunload

    用户正在离开时触发的事件。在这个事件我们可以进行一些询问是否退出的提示。

    # unload

    用户几乎已经离开。
