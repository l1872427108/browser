    # 跨域策略

    一个网站的脚本无法访问其他网站的内容。一个源无法获取另一个源的内容。

    要允许跨源访问, script 标签需要具有 crossorigin 特性。并且远程服务器必须提供特殊的 header。

    1. 没有 crossorigin 特性， 禁止访问。
    2. crossorigin="anonymous" 如果服务器的响应带有包含 * 或我们的源（origin）的 header Access-Control-Allow-Origin, 允许访问。
